C++静态成员

Posted on 2021-03-23 In C++

静态成员

定义

在定义前加了static

例：
static int n;//静态成员变量
static void Print();//静态成员函数

性质

  1. 普通成员变量每个对象有各自的一份，而静态成员变量一共就一份，为所有对象共享
  2. sizeof运算符不会计算静态成员变量
  3. 普通成员函数必须具体作用于某个对象，而静态函数并不具体作用于某个对象
  4. 静态成员不用通过对象就能访问
  5. 本质上是全局变量/全局函数
  6. 设置静态成员的目的是将和某些类紧密相关的全局变量和函数写到类里面，看上去像一个整体，易于维护和理解
    例：矩形总面积、矩形总数等

注意点

  1. 必须在定义类的文件中对静态成员变量进行一次初始化
    例：int CRectangle :: nTotal = 0;

  2. 静态成员函数中不能访问非静态成员变量，也不能调用非静态成员函数

  3. 有时调用复制构造函数，而非构造函数，时不会改变静态成员，而消亡时要从析构函数减去静态成员

    • 调用一个以CRectangle类对象作为参数的函数时
    • 调用一个以CRectangle类对象作为返回值的函数时

    解决办法：为CRectangle类写一个复制构造函数

如何访问静态成员

  1. 类名 :: 成员名

  2. 对象名. 成员名

  3. 指针->成员名

  4. 引用. 成员名

一、构造函数

  1. 名字与类名相同，可以有参数（默认无参数），不能有返回值

  4. 一个类可以有多个构造函数

C++ 成员对象和封闭类

封闭类

定义

有成员对象的类叫封闭类（有一个成员是其他类的对象）

例：

注意点

  1. 任何生成封闭类对象的语句，都要让编译器明白，对象中的成员对象是如何初始化的

    具体做法：通过封闭类的构造函数的初始化列表

封闭类构造函数和析构函数的执行顺序

  • 封闭类对象生成时，先执行所有成员对象的构造函数，然后才执行封闭类的构造函数
  • 对象成员的构造函数调用次序和对象成员在类中的说明次序一样，与它们在成员初始化列表中出现的次序无关
  • 当封闭类的对象消亡时，先执行封闭类的析构函数，然后再执行成员对象的析构函数。次序和构造函数调用次序相反
    （先构造的后析构）

C++ 常量对象，常量成员函数

常量对象

定义对象前加const

常量成员函数

在类的成员函数说明后加const

性质

  1. 执行期间不应修改其所作用的对象
  2. 常量成员函数中不能修改成员变量的值（除静态成员变量），也不能调用同类的非常量成员函数（除静态成员函数）

例

常量成员函数的重载

名字和参数表都一样，一个是const一个不是，算重载

  • 常量对象就调用const成员函数
  • 非常量对象调用非常量成员函数

常引用

  • 对象的引用作为函数的参数，避免复制构造函数的调用。
  • 为避免在函数中更改对象，用常对象引用作为参数
