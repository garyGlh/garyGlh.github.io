Python 数据类型

整型

整型和长整型不严格区分（一般不会溢出）

布尔型

rue或False（大写））

浮点型

可以直接科学记数法e

复数型

a+bj（虚数部分必须有j）

  • 复数的实部：复数.real
  • 复数的虚部：复数.imag
  • 复数的共轭：复数.conjugate()

序列

序列通用

序列类型运算符

  • 下标

    • 负数表示从最后一个算起，最后一个是-1，第一个是-n
    • week[1:4]：取出下标1-3（4-1）的元素
    • week[:6}：取出下标0-5（6-1）的元素
    • week[::-1]：把原序列逆序

  • 重复
    “apple”*3->”appleappleapple”

  • 拼接
    “pine”+”apple”->”pineapple”
  • 包含
    “BA” in (“BA”, “CD”)->True

序列类型转换函数

  • list(x)
  • str(x)
  • tuple(x)

例：

序列类型常用内建函数

  • enumerate(list, start)：返回enumerate对象，是由索引和元素构成的元组，index平移到从start开始
  • len(x)：字符串长度
  • max(x)
  • minx()
  • reversed(x)：逆序
  • sorted(x)：排序，不改变原序列
  • x.sort(reverse=False)：排序，改变原序列
  • sum(x)
  • zip(x)：把对象打包成元组

字符串

单引号，双引号，三引号（多行）均可

  • 可通过+连接

  • 可用r避免转义：r"c:\Users\11546"

列表list（元素可变）

  • .pop(index=n-1)：去掉索引为index的元素（默认是最后一个）
  • .append(x)：最后加上元素x
  • .extend(x)：合并列表x
  • .copy()：浅拷贝
  • .deepcopy()：【copy模块】深拷贝

字典（映射）

用大括号{}界别，成员都是键值对应的形式

  • 形如：d={'sine:'sin','PI':3.14}

创建字典

  • 直接
    aInfo={"xiaoming":300, "xiaoha": 400}
  • dict函数（元组的列表、列表的列表或元组的元组）

设置默认值

  • .fromkeys(所有key, value默认值)
    例：

生成字典

  • zip(list1, list2)：打包（拉链咬合）

基本操作

  • 根据键取值（不推荐）
    d['sine']，没有会出现异常
  • 用.get(key)方法取值（推荐）
    d.get("sine")，没有返回None
  • 添加新成员
    d["mayun"]=100
  • 成员判断
    "mayun" in d
  • 删除成员
    del d["mayun"]
  • 清空字典
    • aStock={}，不会清空引用对象
    • aStock.clear()，会清空引用对象

函数

  • sorted(dict)：返回列表，排序默认是key内部存储顺序
    • 按照key排序：sorted(dict.items(), key=lambda d: d[0]
    • 按照value排序：sorted(dict.items(), key=lambda d: d[1]
    • 先按照value排序，再按照key排序：sorted(dict.items(), key=lambda d: (d[1], d[0])
  • len(dict)：字典元素个数
  • hash(key)：判断某个对象是不是可哈希的（可变的）

方法

  • .keys()：所有key，返回列表
  • .values()：所有value，返回列表
  • .items()：把字典每对key-value组成一个元组，放到列表中（多用于for循环）
  • .update(other_dict)：把other_dict中的更新合并到当前字典

关系运算（运算符可复合）

  • |：并
  • -：-或\
  • ^：对称差

  • .remove(element)：删除某个元素
  • .update(sequence)：增加新元素（序列中的每个元素）
  • .clear()：清空

不可变集合

用frozenset(序列)创建frozenset

数组

array模块

  • 通过array函数创建数组
    array.array("B", range(5))
  • 提供append、insert和read等方法
  • 不支持多维数组

ndarray

  • 所有元素是同一种类型
  • 别名为array
  • 利于节省内存和提高CPU计算
  • 有丰富的函数

基本属性

  • ndarray.ndim：秩（轴的个数），如2
  • ndarray.shape：维度（轴），如(2, 2)
    • axis=0：对每一列操作
    • axis=1：对每一行操作
  • ndarray.size：元素个数
  • ndarray.dtype：元素类型
  • ndarray.itemsize：元素字节大小

  • .linalg.det(array)：行列式

  • .linalg.solve：多元一次方程求根
  • .linalg.elg：求特征值和特征向量
