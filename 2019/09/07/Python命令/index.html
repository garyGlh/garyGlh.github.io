Python命令

发表于 2019-09-07 分类于 Python

Python 命令

  • dir(数据类型/模块)：查看该类型所有方法
  • help(数据类型.函数名/模块)：查看函数用法
  • id(对象）：查看对象身份id
  • import 模块，调用函数时模块.函数

命令行

  • dir：查看当前目录文件夹
  • python -m pip list：查看已安装模块
  • python -m pip install 模块名：按照模块
  • python test.py 3：用sys.argv[1]获取3
