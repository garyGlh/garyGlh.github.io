Python模块、包和库

发表于 2019-10-27 分类于 Python

模块，包和库

模块

导入模块

  • import 模块

  • 导入多个模块：
    用“，”分隔各个模块

  • 导入模块指定属性
    from 模块 import 属性

常用模块

  • random：随机数
    • randint：随机整数

包

  • 一个有层次的文件目录结构
  • 定义了一个由模块和子包组成的Python应用程序执行环境
  • 导入模块：import 包名.子包名.模块

库

  • 一组具有相关功能的模块的集合
  • 导入库里指定模块
    from 库 import 模块

扩展库

SciPy

SciPy中的数据结构
  • ndarray（N维数组）
  • Series（变长字典）
  • DataFrame（数据框）
SciPy核心库

特征：

  • python中科学计算程序的核心包
  • 有效计算numpy矩阵，让NumPy和SciPy协同工作
  • 致力于科学计算中常见问题的各个工具箱，使不同子模块有不同的应用，如插值、积分、优化和图像处理等

NumPy

特征：

  • 强大的ndaaray对象和ufunc函数
  • 精巧的函数
  • 适合线性代数、随机数处理等科学计算
  • 有效的通用多维数据，可定义任意数据类型
  • 无缝对接数据库

random模块
  • .random.normal(mean, var, num)：采样正态分布数据
  • .random.uniform(a, b, num)：采样均匀分布数据
  • 有放回采样
  • 无放回采样（replace=False）
Matplotlib

特征：

  • 基于NumPy
  • 二维绘图库，简单快速地生成曲线图、直方图和散点图等
  • 常用的pyplot是一个简单提供类似Matlab接口的模块
pandas

特征：

  • 基于SciPy和NumPy
  • 高效的Serirs和DataFrame数据结构
  • 强大的可扩展数据操作与分析的python库
  • 高效处理大数据集的切片等功能
  • 提供优化库功能读写多种文件格式，如csv、hdfs
