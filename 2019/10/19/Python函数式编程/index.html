Python 函数式编程

主要由3个基本函数和1个算子构成
基本函数：

  • map()：接收函数和一个列表，把函数逐一映射
  • reduce()：和map很像，但是是递归地计算
  • filter()：筛选出列表中满足条件的元素

算子：

  • lambda
