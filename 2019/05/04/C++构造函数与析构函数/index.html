C++ 构造函数与析构函数

一、构造函数

  1. 名字与类名相同，可以有参数（默认无参数），不能有返回值

  2. 作用是对对象初始化，给成员变量赋初值

  3. 对象生成时构造函数自动被调用，生成后无法再在其上执行构造函数

  4. 一个类可以有多个构造函数

二、复制构造函数

定义：只有一个参数，即同类对象的引用（若没有定义则生成默认复制构造函数，复制）

形如：X::X( X&) 或 X::X( const X&) //必须引用

例：

用复制构造函数的情况用一个对象初始化同类的另一个对象若某函数有一个参数是类A的对象（除非引用），那么该函数被调用时，A的复制构造函数将被调用若函数的返回值是A的对象，则函数返回时，A的复制构造函数被调用（注：对象间赋值不调用复制构造函数，如c2 = c1）
