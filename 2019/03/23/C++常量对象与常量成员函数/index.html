C++常量对象与常量成员函数

发表于 2019-03-23 分类于 C++

C++ 常量对象，常量成员函数

常量对象

定义对象前加const

常量成员函数

在类的成员函数说明后加const

性质

  1. 执行期间不应修改其所作用的对象
  2. 常量成员函数中不能修改成员变量的值（除静态成员变量），也不能调用同类的非常量成员函数（除静态成员函数）

例

常量成员函数的重载

名字和参数表都一样，一个是const一个不是，算重载

  • 常量对象就调用const成员函数
  • 非常量对象调用非常量成员函数

常引用

  • 对象的引用作为函数的参数，避免复制构造函数的调用。
  • 为避免在函数中更改对象，用常对象引用作为参数
