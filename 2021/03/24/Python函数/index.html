内建函数

查找内建函数：dir(两个下划线builtins两个下划线)

数据类型

  • type(x)：数据类型
  • isinatance(x)：数据类型（还可以看是否属于子类，比type强）
  • eval(x)：执行一个字符串表达式，并返回表达式的值，多于input搭配。

数值型内建函数

实用函数

  • input(x)：输入，a=input(“请输入：”)
  • str(x）
  • min(x)/max(x)：找最小/大值，若x是元组，则只看第一位
    • min(x, key=lambda x:x[1])：从元组的序号1找

数学

  • round(x)：四舍五入
  • range(start, end, step=1)：产生等差数列（不包括end），返回可迭代的range对象，可用list(range())转换成列表
    • lazy list，用多少算多少
  • .strip()：去掉空白及换行符

字符串

  • str(int)：int转string
  • repr(int)：int转string，以python表达式形式表现
  • len(str)：字符串长度
  • max/min(str)：最大/最小字符
  • encode(code)：编码
  • decode(code)：解码
  • isalpha(ch)：判断是不是字母

大小写

  • title(x)：每个单词首字母大写
  • upper(x)：全部大写
  • lower(x)：全部小写

空白

  • strip()：删除字符串前后的空白
  • lstrip()：删除字符串前面的空白
  • rstrip()：删除字符串后面的空白

子串

  • startswith(子str)：判断字符串是否以指定子串开头
  • endswith(子str)：判断字符串是否以指定子串结尾
  • find(子str, start, end)：在[start, end)查找子串在字符串中出现位置，没找到返回-1
  • index(子str)：查找子串在字符串中出现位置，没找到引发错误
  • replace(子str)：用指定字串替换字符串中目标子串
  • translate(子str)：使用指定翻译映射表替换字符串

分割、连接

  • split(deperator=””)：分割字符串，不改变原始字符串，返回列表
  • join(连接符，字符串列表)：将多个单词连成字符串

替换

  • replace(ori_str, new_str)：把ori_str替换为new_str

进制转换（返回值为str）

  • hex(int)：将整数转化为16进制
  • oct(int)：将整数转化为8进制
  • bin(int)：将整数转化为2进制

标准库函数

math

  • pi：π
  • e
  • floor(x)：向下取整
  • ceil(x)：向上取整
  • pow(x, y)：x的y次方
  • degrees(x)：弧度转角度
  • radians(x)：角度转弧度

os

  • getcwd()：获得当前目录
  • chdir(path)：转到目录path
  • rename(origin, new)：把origin文件重命名为new
  • remove(file)：删除文件file
  • listdir(path)：当前文件夹文件名列表
  • .path.join(path, filename)：把文件路径和文件名连接
  • .path.exists(path)：判断目录是否存在
  • mkdir(path)：创建目录

shutil

  • rmtree(path)：递归地删除非空目录

random

  • choice([a, b, …])：从序列中获得随机值
  • randint(x, y)：生成[x, y]（包含y）之间的随机整数
  • randrange(start, end step)：从range中获取随机整数
  • random()：生成0-1.0的随机浮点数
  • uniform(5, 10)：生成5-10的随机浮点数
  • sample(range(a,…), num)：从range中获得num个随机数，以列表形式返回
  • shuffle(nums) ：将列表nums中的元素打乱

datetime

date模块

  • today()：今天日期

time模块

  • time(h, min, sec)：定义一个时间

datetime模块

  • now()：获得当前日期+时间+ms
    • 可用print(dt. strftime("%a %b %d %Y %H: %M %S")形式转换成字符串
  • datetime(y, m, d, h, min, sec)：定义一个日期+时间
  • timestamp()：将本地时间和日期转为时间戳
  • fromtimestamp(ts)：将时间戳ts转为本地时间和日期
