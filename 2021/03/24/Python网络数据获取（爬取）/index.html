Python网络数据获取（爬取）

Posted on 2021-03-24 In Python

Python 网络数据获取（爬取）

  • 抓取

    • urlib内建模块（被取代）

      • urlib.request

    • Requests第三方库（中小型）

    • Scapy框架（大型）

  • 解析

    • BeautifulSoup库
    • re模块（正则表达式）

  • 或用第三方API抓取+解析

抓取

Requests库

基本方法：

  • r=requests.get(url)：请求获取指定URL位置的资源，对应HTTP协议的GET方法
  • r.status_code：查看状态码（200为正常）
  • r.encoding=...：修改编码
  • r.text：查看内容

解码

  • 二进制：r.content
  • 文本：r.text
  • json：r.json()
