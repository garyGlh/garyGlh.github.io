发表于 2021-03-24 分类于 R

R 数据类型

vector

同类型序列

定义

seq()

rep()

rdist()

查看分布的指令：?Distributions

可存储数据类型

  • logical逻辑：TRUE, FALSE
  • numeric数值（float或int）：5, 3.14
  • integer整数：2L, 34L（后面要加L表示是整数）
  • complex复数：3+2i
  • character字符或字符串：”Hello”, ‘a’

操作

类型转换

  • 默认全部转为character型：vtr1=c(“hi”, 3, TRUE)

数量操作（每个元素）

+, -, ^, *, %*%（数量积）

比较（每个元素）

访问元素

  1. 访问某个位置的元素：vtr[2]（下标从1开始，而非从0开始）
  2. 访问某个范围的元素：vtr[2:6]（闭区间）
  • 若插入元素越界，则在有元素之后到新访问的元素之间都显示NA
  • replace多个元素：X[2:3]=c(22, 33)

  1. 排除该位元素：下标为负

  2. 取多个位置的元素：下标为c(a, b)

  3. 增加元素：

函数

一般函数
  • 排序：sort(X)
  • 长度：length(X)
  • 范围：range(X)
  • 取前n个元素：head(X, n=5)
  • 取后n个元素：tail(X, n=5)
  • 展示vector类型及元素：str(X)
数学函数（整体）
  • 均值：mean(X)
  • 标准差：sd(X)
  • 方差：var(X)
  • 最大值/最小值：max(X)/min(X)
  • 中位数：median（X）
  • 求和：sum(X)
  • 求积：prod(X)
  • 百分比：quantitlle(X, probs=0.5)
  • 均值及四个分位点：summary(X)
数学函数（每个元素）
  • log(X)
  • exp(X)
  • sqrt(X)

list

不同类型元素构成的序列

  • list不能排序

例

x=list(“hi”, 3, TRUE)

操作

函数

  • 合并两个list：
    • merge（list1, list2）
    • list3=c(list1, list2)

array

存储两维及以上的数据，通常以vector作为输入

定义

result=array(c(vtr1, vtr2), dim=c(3, 3, 2))

  • dim是维

matrice

二维

定义

x=matrix(data, nrow, ncol, byrow, dimnames)

  • byrow是逻辑：默认FALSE（按列）

例

factor

将元素按等级分类

定义

fct=factor(vtr1)

data frame

变量属性表（多个变量，多个属性）

定义

x=data.frame(属性1，属性2）

  • 不同属性的元素数量一定要相同

例

函数

  • 展示：str(X)
