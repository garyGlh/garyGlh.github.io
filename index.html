例题

爬楼梯

问题：n阶楼梯，每次可爬1或2层，求所有方法数

  • dp[i]表示爬到i阶楼梯的方法数

  • dp[i]=dp[i-1]+dp[i-2]

除数博弈

问题：A和B轮流选数字，每次可选N%k==0的k，另一人在N-k里选，问能否赢

  • dp[i]为布尔数组，表示如果当前选择范围为0~i，则在自己与对手均以最佳状态参赛时将胜利/失败

        if(p[i-j]==false)//对手输的方法，找到则break，找不到则最终false
        {
            p[i]=true;
            break;
        }

R 数据类型

vector

同类型序列

定义

seq()

rep()

rdist()

查看分布的指令：?Distributions

可存储数据类型

  • logical逻辑：TRUE, FALSE
  • numeric数值（float或int）：5, 3.14
  • integer整数：2L, 34L（后面要加L表示是整数）
  • complex复数：3+2i
  • character字符或字符串：”Hello”, ‘a’

操作

类型转换

  • 默认全部转为character型：vtr1=c(“hi”, 3, TRUE)

数量操作（每个元素）

+, -, ^, *, %*%（数量积）

比较（每个元素）

访问元素

  1. 访问某个位置的元素：vtr[2]（下标从1开始，而非从0开始）
  2. 访问某个范围的元素：vtr[2:6]（闭区间）
  • 若插入元素越界，则在有元素之后到新访问的元素之间都显示NA
  • replace多个元素：X[2:3]=c(22, 33)

  1. 排除该位元素：下标为负

  2. 取多个位置的元素：下标为c(a, b)

  3. 增加元素：

函数

一般函数
  • 排序：sort(X)
  • 长度：length(X)
  • 范围：range(X)
  • 取前n个元素：head(X, n=5)
  • 取后n个元素：tail(X, n=5)
  • 展示vector类型及元素：str(X)
数学函数（整体）
  • 均值：mean(X)
  • 标准差：sd(X)
  • 方差：var(X)
  • 最大值/最小值：max(X)/min(X)
  • 中位数：median（X）
  • 求和：sum(X)
  • 求积：prod(X)
  • 百分比：quantitlle(X, probs=0.5)
  • 均值及四个分位点：summary(X)
数学函数（每个元素）
  • log(X)
  • exp(X)
  • sqrt(X)
